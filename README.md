# soccer-program-in-c

This interactive c program collects results of soccer teams in a league and writes the report to an external file.
The game statistics collected include: 
-> total number of points from each team after the league;
-> Total number of goals scored by each team;
-> Total number of wins from each team;
-> Total number of draws from each team;
-> Total number of losses from each team.

Clone the source code and modify it to your preference, Happy Coding!
